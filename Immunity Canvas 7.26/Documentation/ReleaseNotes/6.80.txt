This release comes with one new exploit module, for the recent PHP-CGI
remote vulnerability as described in CVE-2012-1823. Moreover, we include
two new utility modules, a generic WAR MOSDEF trojan deployer and an IP->VHOST

completely new user interface and the MOSDEF OSX remote resolvers updated for
the latest OSX versions.
