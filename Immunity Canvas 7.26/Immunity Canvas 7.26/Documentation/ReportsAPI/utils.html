utils¶

class libs.reports.utils.Data(*args, **kwargs)¶

A dict subclass that allows attribute access to values.

Attributes can be accessed directly or using the standard dict API. Any attribute/value that is missing will return None.

class libs.reports.utils.DefaultData(*args, **kwargs)¶

A Data subclass that returns <unknown> instead of None.

This class is useful to pass to templates so that values will display <unknown> rather than None.
