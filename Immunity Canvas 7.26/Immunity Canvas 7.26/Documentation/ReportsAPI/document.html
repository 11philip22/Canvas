class libs.reports.document.ReportDocument(filename)¶

odfpy wrapper for the XML files in an OpenDocument.

odfpy provides a high-level API that gives you full control to edit every aspect of an OpenDocument. This class wraps that API and provides some helper methods and some CANVAS specific utils.
